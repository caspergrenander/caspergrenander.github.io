About the AutoPublisher Tool

AutoPublisher is a manual publishing tool that enables authenticated TikTok users to upload and publish their own video content to their own TikTok account via TikTok's official Content Posting API.

The Direct Post functionality is used solely to allow a logged-in user to manually publish a selected video file directly from the web interface to their own TikTok account.

All actions are initiated explicitly by the user after logging in via TikTok's OAuth system. The tool does not support bulk posting, scheduling, automated posting, or background publishing.

The tool is used individually per authenticated TikTok account and no user data is shared between accounts.
