Data Deletion

Historien Bakom Musiken AutoPublisher uses TikTok's official authentication system (OAuth) to temporarily access the user's account for the purpose of uploading and publishing video content.

The application stores no personal data in any permanent database. Only authentication tokens (access token and refresh token) are handled in the server environment to enable user-initiated actions.

What Gets Deleted

When you request data deletion, the following will be removed:

  • All authentication tokens associated with your account
  • Any temporary file references in the service
  • All session data associated with your usage

The process is usually completed within 7-10 business days after the request is received.
