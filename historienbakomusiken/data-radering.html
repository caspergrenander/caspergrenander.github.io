Data Radering

Senast uppdaterad: 2025-01-01

Vad kan raderas

AutoPublisher lagrar inte videofiler eller personuppgifter lokalt i systemet. Systemet använder temporära autentiseringstokens som tillhandahålls av TikTok för att möjliggöra uppladdning och publicering. Följande typer av uppgifter kan hanteras:

  • Tillfälliga autentiseringstokens från TikTok som används för sessioner vid uppladdning.
  • Teknisk metadata som behövs för att hantera uppladdningsförfrågningar i realtid.

Hur du begär radering

För att begära radering av data som hanteras i samband med användning av Tjänsten, skicka ett e-postmeddelande till:

E-post: casper.grenander5@gmail.com

I din begäran ange:

  • Ditt TikTok-användarnamn eller annan identifiering som användes vid inloggning
  • Vilken typ av åtgärd du önskar (till exempel radering eller åtkomstförfrågan)
  • Kontaktuppgifter för bekräftelse

Callback URL för TikToks Data Deletion

Om TikTok initierar en formell raderingsförfrågan använder Tjänsten följande publika callback-URL:

https://caspergrenander.github.io/historienbakomusiken/data-radering

Tidsram

När en raderingsbegäran mottagits kommer vi att bekräfta mottagandet via e-post och påbörja raderingsprocessen. Då en del data hanteras av TikTok kan slutlig radering ta längre tid, beroende på TikToks egna rutiner. Vi strävar efter att slutföra vår del inom 7 kalenderdagar.
