Data Radering

Senast uppdaterad: 2025-01-01

Vad kan raderas

AutoPublisher lagrar inte videofiler eller personuppgifter lokalt i systemet. Systemet använder temporära autentiseringstokens som tillhandahålls av TikTok för att genomföra uppladdningar. Följande typer av uppgifter hanteras eller kan påverkas:

  • Tillfälliga autentiseringstokens från TikTok som används för sessioner vid uppladdning.
  • Metadata och loggar som behövs för att genomföra och spåra uppladdningsförfrågningar.

Hur du begär radering

För att begära radering av data som hanteras i samband med användning av Tjänsten, skicka ett e-postmeddelande till:

E-post: casper.grenander5@gmail.com

I din begäran ange:

  • Ditt TikTok-användarnamn eller den identifiering som du använt vid inloggning
  • Vilken typ av åtgärd du önskar (t.ex. radering, åtkomstbegäran)
  • Kontaktuppgifter för bekräftelse

Callback URL för TikToks Data Deletion

Om TikTok initierar en formell data-deletion-begäran använder vi följande publika URL som callback för verifiering:

https://caspergrenander.github.io/historienbakomusiken/data-radering

Tidsram

När en raderingsbegäran mottagits kommer vi att bekräfta mottagandet via e-post och påbörja raderingsprocessen. Eftersom viss data hanteras av TikTok kan slutlig radering ta längre tid beroende på TikToks egna processer. Vi strävar efter att bekräfta och slutföra raderingen inom 7 kalenderdagar när det är möjligt.
