1. Tillämpning

Dessa användarvillkor ("Villkoren") gäller för din användning av Historien Bakom Musiken AutoPublisher (”Tjänsten”). Genom att logga in och använda Tjänsten godkänner du dessa villkor. Tjänsten erbjuds som ett creator-verktyg och är tillgänglig för användare som autentiserar sig via TikToks officiella OAuth-system.

2. Vad Tjänsten gör

AutoPublisher möjliggör uppladdning och publicering av videoklipp till ett TikTok-konto via TikToks Content Posting API. Användaren initierar inloggning via TikTok, väljer material och styr publiceringen genom Tjänstens gränssnitt. Tjänsten automatiserar inte publicering utan användaren bekräftar uppladdning/publicering i det normala användarflödet.

3. Åtkomst och autentisering

För att använda Tjänsten krävs en TikTok-inloggning via OAuth. Endast de scopes som är nödvändiga för funktionaliteten begärs (user.info.basic, video.upload, video.publish). Användaren kan när som helst återkalla åtkomst via sina TikTok-kontoinställningar.

4. Ansvar för innehåll

Användaren ansvarar för allt innehåll som laddas upp och publiceras via Tjänsten. Du bekräftar att du har rätt att publicera materialet och att det inte bryter mot upphovsrätt, personliga rättigheter eller tillämplig lag. Utvecklaren kan inte hållas ansvarig för innehåll som publiceras av användaren.

5. Begränsningar

  • Tjänsten får inte användas för att kringgå TikToks regler eller för att automatisera masspublicering i strid med plattformens policy.
  • Användning i kommersiella tredjepartstjänster eller vidareförsäljning av Tjänsten är inte tillåten utan skriftligt avtal med utvecklaren.
  • Utvecklaren förbehåller sig rätten att stänga av åtkomst för användare som missbrukar Tjänsten eller bryter mot dessa villkor.

6. Personuppgifter och säkerhet

AutoPublisher begär endast de behörigheter som krävs för funktionaliteten. Ingen användardata eller tokens sparas permanent i systemet utöver kortlivade autentiseringsuppgifter som krävs för att genomföra en uppladdning. All kommunikation sker över krypterade anslutningar (HTTPS).

7. Dataradering

Instruktioner för begäran om radering finns på sidan för Data Radering: Data Radering. Observera att en del data hanteras av TikTok och slutlig radering kan kräva åtgärd från TikTok.

8. Ansvarsbegränsning

I den utsträckning lagen tillåter är utvecklarens ansvar begränsat till direkta skador som orsakats av grov vårdslöshet eller uppsåt. Utvecklaren ansvarar inte för indirekta skador, förlorad vinst eller följdskador orsakade av användning av Tjänsten.

9. Ändringar av villkoren

Dessa villkor kan uppdateras vid behov. Vid väsentliga ändringar meddelas detta via webbplatsen. Den version som är publicerad på webbplatsen vid tidpunkten för din användning gäller.
