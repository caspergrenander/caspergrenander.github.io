1. Tillämpning

Dessa användarvillkor ("Villkoren") gäller för din användning av Historien Bakom Musiken AutoPublisher (”Tjänsten”). Genom att logga in och använda Tjänsten godkänner du dessa villkor. Tjänsten erbjuds som ett creator-verktyg och är tillgänglig för alla användare som autentiserar sig via TikToks officiella OAuth-system.

2. Vad Tjänsten gör

AutoPublisher möjliggör uppladdning och publicering av videoklipp till ett TikTok-konto via TikToks Content Posting API. Användaren initierar inloggning via TikTok, väljer material och styr publiceringen genom Tjänstens gränssnitt. Tjänsten automatiserar inte publicering utan användaren bekräftar själv alla steg.

3. Åtkomst och autentisering

För att använda Tjänsten krävs en TikTok-inloggning via OAuth. Endast de scopes som är nödvändiga för funktionaliteten begärs (user.info.basic, video.upload, video.publish). Användaren kan när som helst återkalla åtkomst via sina TikTok-inställningar.

4. Ansvar för innehåll

Användaren ansvarar för allt innehåll som laddas upp och publiceras via Tjänsten. Du bekräftar att du har rätt att publicera materialet och att det inte bryter mot upphovsrätt, personliga rättigheter eller lag. Utvecklaren ansvarar inte för innehåll som publiceras av användaren.

5. Begränsningar

  • Tjänsten får inte användas för att kringgå TikToks regler, villkor eller tekniska begränsningar eller för otillåten automatisering.
  • Tjänsten får inte användas i kommersiella tredjepartstjänster utan avtal.
  • Utvecklaren kan stänga av åtkomst vid missbruk.

6. Personuppgifter och säkerhet

AutoPublisher lagrar inga personuppgifter i någon extern eller kommersiell databas. Autentisering sker via TikToks OAuth-system och access tokens samt refresh tokens hanteras säkert i servermiljö enbart för att upprätthålla användarens autentiserade session och genomföra användarinitierade åtgärder.

All kommunikation mellan användaren, tjänsten och TikTok sker via krypterade anslutningar.

7. Dataradering

Instruktioner finns här: Data Radering. Viss data hanteras av TikTok och kan kräva åtgärd från dem.

8. Ansvarsbegränsning

Utvecklarens ansvar är begränsat enligt lag och omfattar inte indirekta skador eller följdskador.

9. Ändringar av villkoren

Villkoren kan uppdateras. Den publicerade versionen gäller vid användningstillfället.
