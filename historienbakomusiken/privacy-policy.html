Privacy Policy

Last updated: 2025-01-01

Historien Bakom Musiken AutoPublisher is a creator tool that enables uploading and publishing of video clips via TikTok's official Content Posting API. This privacy policy describes how data is handled in connection with the use of the service.

Data Collection and Usage

  • The service does not store personal data in any external or commercial database.
  • Authentication occurs via TikTok's OAuth system. Access tokens and refresh tokens are securely stored in the server environment solely for the purpose of maintaining the user's authenticated session.
  • Tokens are used only to perform user-initiated actions and are never shared with third parties.
  • No video files or user content are permanently stored by the service.
  • Only the scopes required for functionality are requested: user.info.basic, video.upload, video.publish.
  • All communication between the user, the service, and TikTok occurs over secure and encrypted connections (HTTPS).

Data Deletion

Users can request deletion of data handled in connection with the use of the service. Since the majority of all data is handled by TikTok, it may require users to also perform deletion actions directly with TikTok.

Users can also revoke access to the service at any time via their TikTok account settings.

Rights and Contact

Users can request information, correction, or deletion of data by contacting the developer via email. Clearly specify the type of request and relevant identification (e.g., TikTok username).

Changes to the Privacy Policy

The policy may be updated as needed. Any changes will be published on this page. The date of the last update is indicated at the top.
