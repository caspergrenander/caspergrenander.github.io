Historien Bakom Musiken AutoPublisher

Historien Bakom Musiken AutoPublisher är ett personligt utvecklingsverktyg skapat och använt av Casper Grenander. Syftet med systemet är att automatiskt publicera videoklipp från podden Historien Bakom Musiken till ett enda TikTok-konto, via TikTok Content Posting API. Systemet är privat, används endast av utvecklaren själv, och är inte tillgängligt för allmänheten.

API-användning och behörigheter

Tjänsten använder endast de behörigheter (scopes) som krävs för att fungera korrekt via TikToks officiella API:

  • user.info.basic – används för att identifiera det autentiserade TikTok-kontot.
  • video.upload – används för att ladda upp videofiler till TikTok.
  • video.publish – används för att publicera uppladdade videoklipp.

Alla interaktioner sker genom TikToks säkra OAuth-flöde och följer TikToks utvecklarvillkor.

Integritetspolicy

  • Systemet lagrar inga personuppgifter utöver de autentiseringstokens som TikTok tillhandahåller vid inloggning.
  • Tokens används endast för att publicera innehåll till det egna TikTok-kontot och sparas inte permanent på externa servrar.
  • Ingen data delas, analyseras eller säljs till tredje part.
  • All kommunikation sker via krypterade anslutningar (HTTPS).
  • Åtkomst kan när som helst återkallas av utvecklaren via TikTok Developers-portalen.

Användarvillkor

  • Tjänsten används endast av utvecklaren själv för privat innehållshantering.
  • Ingen extern åtkomst eller användning är tillåten.
  • All användning sker i enlighet med TikToks API-regler, användarvillkor och tillämplig lagstiftning.
  • Systemet får inte användas för att kringgå TikToks riktlinjer, automatisera olämpligt innehåll eller på annat sätt missbruka plattformen.

Ändringar av policy

Denna policy kan uppdateras vid behov, exempelvis vid förändringar i TikToks API eller utvecklarvillkor. Datum för senaste uppdatering anges alltid högst upp på sidan.
