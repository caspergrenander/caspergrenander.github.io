About the Project

Historien Bakom Musiken is a creator-driven project where I, Casper Grenander, share insights from music creation, podcast production, and the creative process. The project aims to provide transparency, inspiration, and context about how content is created and published.

About the AutoPublisher Tool

AutoPublisher is a tool that enables uploading and publishing video clips to a TikTok account via TikTok's official Content Posting API.

All actions are initiated explicitly by the user after logging in via TikTok's OAuth system. The tool does not perform any automatic or unattended publications.

The tool is used individually per authenticated TikTok account and no user data is shared between accounts.
