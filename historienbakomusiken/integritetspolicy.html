Integritetspolicy

Senast uppdaterad: 2025-01-01

Historien Bakom Musiken AutoPublisher är ett creator-verktyg som möjliggör uppladdning och publicering av videoklipp via TikToks officiella Content Posting API. Denna integritetspolicy beskriver hur data hanteras i samband med användning av tjänsten.

Insamling och användning av data

  • Tjänsten samlar inte in och lagrar inte personuppgifter i någon permanent databas.
  • Autentisering sker via TikToks OAuth-system och tokens lagras endast temporärt under en aktiv session.
  • Inga videofiler, tokens eller användaruppgifter sparas permanent på servern.
  • Endast de scopes som krävs för funktionaliteten begärs: user.info.basic, video.upload, video.publish.
  • All kommunikation mellan användaren, tjänsten och TikTok sker över säkra och krypterade anslutningar (HTTPS).

Dataradering

Användare kan begära radering av data som hanteras i samband med användning av tjänsten. Eftersom majoriteten av all data hanteras av TikTok kan det krävas att användaren även genomför raderingsåtgärder direkt hos TikTok.

Rättigheter och kontakt

Användare kan begära information, rättelse eller radering av data genom att kontakta utvecklaren via e-post. Ange tydligt vilken typ av förfrågan som avses samt relevant identifiering (till exempel TikTok-användarnamn).

Ändringar av Integritetspolicyn

Policyn kan uppdateras vid behov. Eventuella ändringar publiceras på denna sida. Datum för senaste uppdatering anges högst upp.
