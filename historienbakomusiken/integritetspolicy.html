Integritetspolicy

Senast uppdaterad: 2025-01-01

Historien Bakom Musiken AutoPublisher är ett creator-verktyg som möjliggör uppladdning och publicering av videoklipp via TikToks officiella Content Posting API. Verktyget kan användas av alla användare som loggar in och autentiserar sig via TikToks OAuth-system. Denna policy beskriver hur data hanteras i samband med användning av appen.

Insamling och användning av data

  • Appen samlar inte in personuppgifter för att skapa en användardatabas.
  • Autentiseringstokens tillhandahålls direkt av TikTok och används endast temporärt för att möjliggöra inloggning och publicering.
  • Inga tokens, videofiler eller användaruppgifter sparas permanent i systemet.
  • Appen använder endast de scopes som krävs för funktionaliteten: user.info.basic, video.upload och video.publish.
  • All trafik mellan användaren, tjänsten och TikTok sker via krypterade anslutningar (HTTPS).

Dataradering (TikTok Data Deletion Request)

Användare som vill begära radering av data som hanteras i samband med appens användning kan kontakta utvecklaren via nedanstående uppgifter. Observera att majoriteten av data lagras av TikTok och att raderingsprocessen kan kräva åtgärder från TikTok direkt.

Rättigheter och kontakt

Förfrågningar om information, rättelse eller radering av data skickas via e-post. Ange tydligt vilken dataförfrågan avser och vilken åtgärd som önskas.

Ändringar av denna policy

Policyn kan uppdateras vid behov. Datum för senaste uppdatering anges alltid högst upp på sidan. Väsentliga ändringar kommuniceras via webbplatsen.
