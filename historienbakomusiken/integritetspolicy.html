Integritetspolicy

Senast uppdaterad: 2025-01-01

Historien Bakom Musiken AutoPublisher är ett creator-verktyg som gör det möjligt för användare att publicera videoklipp till TikTok via TikToks officiella API. Verktyget kan användas av alla som loggar in och autentiserar sig via TikToks OAuth-system. Denna policy beskriver hur appen hanterar data och vilka rättigheter användare har.

Insamling och användning av data

  • Appen samlar inte in personuppgifter i form av en lokal användardatabas.
  • Endast de autentiseringstokens som TikTok tillhandahåller används temporärt för att möjliggöra inloggning och publicering av videor.
  • Inga tokens, videofiler eller användaruppgifter sparas permanent eller delas med tredje part.
  • Appen använder endast de nödvändiga scopes som krävs för funktionaliteten: user.info.basic, video.upload och video.publish.
  • Alla API-anrop och all datakommunikation sker via säkra, krypterade anslutningar (HTTPS).

Dataradering (TikTok Data Deletion Request)

Användare som vill begära radering av data kopplad till autentisering eller videopostning via denna app kan kontakta utvecklaren via nedanstående adress. Observera att TikTok hanterar majoriteten av datan direkt i sina egna system.

Rättigheter och kontakt

Användare kan begära tillgång till lagrad data, rättelse eller radering genom att kontakta utvecklaren via e-post. Specificera tydligt vilken åtgärd som avses.

Ändringar av denna policy

Policyuppdateringar kan utföras vid behov, exempelvis när TikTok uppdaterar sina API-regler. Datum för senaste uppdatering anges högst upp på sidan.
