1. Allmänt

Den här tjänsten, Historien Bakom Musiken AutoPublisher, gör det möjligt att autentisera ett TikTok-konto och publicera videor via en automatiserad process. Genom att använda tjänsten godkänner du dessa villkor.

2. Användning av TikToks API

Tjänsten använder TikToks officiella API för autentisering och publicering. Alla interaktioner följer TikToks utvecklarpolicyer. Du ger tjänsten tillåtelse att utföra de handlingar du uttryckligen godkänner genom TikToks OAuth-flöde.

3. Ansvar

Du ansvarar för att det material som publiceras via tjänsten följer TikToks regler och lokala lagar. Tjänsten tar inget ansvar för borttagna, begränsade eller nedstängda konton.

4. Begränsningar

Tjänsten är avsedd för manuellt godkända automationsflöden och får inte användas för missbruk, spam eller otillåten datainsamling.

5. Ändringar av villkor

Villkoren kan uppdateras vid behov. Vid större ändringar publiceras ett datum för senaste uppdatering högst upp på sidan.
