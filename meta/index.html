About the Automation Tool

C / A Beats Publisher enables scheduled publishing of video content to Facebook and Instagram via Meta's official Graph API.

All actions are initiated explicitly by authorized users after authentication via Meta's OAuth system. The system does not perform any automated actions without user authorization.

The tool is used internally within C / A Beats & Productions and no user data is shared with external parties.

Authentication via Meta

Meta's OAuth authentication is used to verify user identity and enable publishing functionality. Only necessary permissions are requested:

  • pages_manage_posts ‚Äì Manages posts on connected pages
  • pages_read_engagement ‚Äì Reads page engagement data
  • instagram_basic ‚Äì Basic Instagram functionality
  • instagram_content_publish ‚Äì Publishes to Instagram

No personal data is stored permanently. Authentication tokens are used only to carry out user-initiated actions and are handled according to Meta's security requirements.
