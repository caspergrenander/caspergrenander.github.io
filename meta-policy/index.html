Integritetspolicy & Användarvillkor

C / A Beats Publisher – Facebook & Instagram Automation

Detta system används internt för schemalagd publicering av videoinnehåll till Instagram och Facebook via Meta Graph API. Systemet är endast avsett för intern användning av C / A Beats & Productions.

Integritetspolicy

  • Systemet samlar inte in eller lagrar personuppgifter från externa användare.
  • Systemet lagrar endast tekniska åtkomsttokens kopplade till våra egna företagskonton.
  • Inga data analyseras, delas eller säljs till tredje part.
  • Alla anslutningar hanteras via säker OAuth-autentisering.
  • Åtkomsttokens kan när som helst återkallas av Meta eller av ansvarig systemadministratör.

Vilken data hanteras?

Systemet hanterar enbart den metadata som krävs för att publicera innehåll via API, t.ex.:

  • Filnamn och videostorlek
  • Bild- eller videofiler som användaren själv laddar upp
  • Tid för schemalagd publicering
  • Publiceringsresultat (t.ex. status OK eller felmeddelande)

Inga personuppgifter samlas in från andra användare eller följare.

Användarvillkor

  • Systemet får endast användas av behörig personal.
  • Ej avsett för offentlig användning eller kommersiell licensiering.
  • Obehörigt intrång, manipulering eller försök att ändra systemet är förbjudet.
